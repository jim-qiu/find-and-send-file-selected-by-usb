# find-and-send-file-selected-by-usb
1. 前提：
1.1 安装usb，qt5库，在windows里的终端窗口输入：pip install pyusb pyqt5
2. 操作步骤如下
2.1 运行该python
2.2 输入搜索路径和文件名中的关键字，搜索文件，选择某一个文件
2.3 输入usb设备的vid，pid，输入端点地址，输出端点地址，接口号，通过usb发送出去，如下图所示
